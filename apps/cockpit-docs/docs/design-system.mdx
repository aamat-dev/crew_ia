# Design System

## Tokens

- Couleurs, typographie et espacements centralisés.
- Utiliser les variables CSS `var(--token)`.

## Composants

- Composants réutilisables basés sur React.
- Préférer la composition à l'héritage.

## Motion

### À faire
- Transitions sous 200 ms.
- Utiliser `prefers-reduced-motion`.

### À éviter
- Animations infinies.
- Flashs ou mouvements excessifs.
